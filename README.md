## Overview

### Description

A social media for fitness. Allows users to tap in into friends workout sessions, mark personal records, share routines, photos and invite to workout together.

### App Evaluation

[Evaluation of your app across the following attributes]
- **Category:** Fitness Social Media
- **Mobile**: Connect with friends at the gym. Snap a quick photo, share a real-time notification that you are working out, mark on a map if you worked out at a gym or in a park. 
    - **Story**: Make working out a fun, competitive, and bonding experience with friends een if far away. Create a healthy habit and push friends to be better, workout consistently and get in shape together. 
    - **Market**: People who workout, can be weightlifters, combat sports, yoga, running, calisthenics, endurance, sports. From ages 12-80. Anyone who works out can use this app. 
    - **Habit**: Users can interact with the app many times througout the day. Planning, sharing, using a workout. With features like notifications and real time alers, app can become habbit forming.
    - **Scope**: Can start very simple by sharing texts of workout descriptions, having personal records, adding friends. Progress into sharing images, workout locations, calendar invites.

## Product Spec

### 1. User Stories (Required and Optional)

**Required Must-have Stories**

* As an athlete, I want to see how much I have progressed overtime.
* User can login
* As an abroad student, I want to see how my friends are working out.
* As a begginer, I want to know what routines other people are doing.
* As an athlete, I want to see other variations of training for my injury.

**Optional Nice-to-have Stories**

* As a frequent user, I want to know where I am on the leaderboard.
* As a friend, I want to know where my friends are working out.

### 2. Screen Archetypes

- Login
* User can login
* User can signup
- Stream
* User can look at what other users are doing
* User can see recommended workout sessions
- Detail
* Detail about a friends workout and react to it or save it.
* Detail about a workout routine the user can try out and save.
- Creation
* User can create and journal their workout here. Add workouts, reps, sessions, time, rest time, PR.
- Profile
* User can look at their progress, what they have saved, their friends, look at personal records.

### 3. Navigation

**Tab Navigation** (Tab to Screen)

* Home Feed
* React to a session
* Post a session

**Flow Navigation** (Screen to Screen)

- Login Screen
* Home
- Registration Screen
* Home
- Stream Screen
* See reactions to a session.
* Save the workout routine/session
- Creation Screen
* Home
